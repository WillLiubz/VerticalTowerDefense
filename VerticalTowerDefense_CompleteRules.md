# Vertical Defense - 完整玩家规则文档

## 一、游戏概述

**游戏名称**：Vertical Defense
**核心玩法**：竖版塔防 + 卡牌构筑 + 横向支援 + 赌博博弈
**单局时长**：3分钟
**目标用户**：欧美休闲/中度玩家

---

## 二、核心游戏循环

```
击杀怪物 → 获得金币 → 购买卡包/赌博 → 获得/升级卡牌 → 合成强化 → 释放Buff
     ↑                                                              ↓
     └──────────── 阻止怪物到达底部 ← 部署支援角色 ← 升级武器 ←─────┘
```

## 三、基础塔防规则

### 3.1 战场布局

```
┌─────────────────────────────────────────┐
│              怪物生成区 (Spawn)           │  ← 波次开始处
├─────────────────────────────────────────┤
│                                         │
│  [支援槽0]        │        [支援槽2]     │  ← 横向支援列
│      →→→         │         ←←←         │
│  [支援槽1]        │        [支援槽3]     │
│      →→→         │         ←←←         │
│                   │                     │
│  ═══════════════════════════════════   │  ← 怪物行进主路径
│                   │                     │
│    [武器0] [武器1] [武器2] [武器3]       │  ← 底部主防线
│                                         │
│  ───────────────────────────────────   │  ← 防线底线
│                                         │
│  [卡包] [赌博]  │  [卡牌槽 x6]           │  ← 操作区
└─────────────────────────────────────────┘
```

### 3.2 胜负条件

| 条件 | 结果 |
|------|------|
| 成功防守10波 (3分钟) | **胜利** |
| 防线生命值归零 | **失败** |
| 怪物突破底线20次 | **失败** |

### 3.3 防线机制

- **初始生命值**：20点
- **扣血规则**：每个怪物到达底线扣除1点
- **无法恢复**：除非使用特定卡牌

## 四、武器系统（底部防线）

### 4.1 武器槽位

- **数量**：4个固定位置
- **更换机制**：战斗中可随时更换武器类型（冷却2秒）
- **升级方式**：通过卡牌Buff提升等级

### 4.2 武器类型详解

| 武器 | 图标 | 伤害 | 射速 | 射程 | 特殊效果 | 最佳用途 |
|------|------|------|------|------|---------|---------|
| **机枪塔** | 🔫 | 8 | 极快(150ms) | 300 | 无 | 清理小怪群 |
| **狙击塔** | 🎯 | 50 | 慢(1200ms) | 500 | **穿透**多个敌人 | 高血量目标 |
| **激光塔** | ⚡ | 3/帧 | 持续 | 250 | 持续伤害 | 直线区域 |
| **火箭塔** | 🚀 | 30 | 中(800ms) | 350 | **范围爆炸** | 密集敌群 |

### 4.3 武器强化

```
基础伤害 × 武器等级 × (1 + 伤害Buff加成)

例：狙击塔 Lv2 + 伤害Buff Lv1
= 50 × 2 × 1.2 = 120伤害/发
```

## 五、卡牌系统

### 5.1 卡包等级机制 ⭐新增

**核心设计**：累计抽卡提升卡包等级，高等级卡包出高等级卡

| 卡包等级 | 累计抽卡次数 | Lv1卡概率 | Lv2卡概率 | Lv3卡概率 |
|---------|-------------|----------|----------|----------|
| **普通** | 0-4次 | 100% | 0% | 0% |
| **高级** | 5-14次 | 70% | 30% | 0% |
| **稀有** | 15-29次 | 50% | 40% | 10% |
| **史诗** | 30+次 | 30% | 50% | 20% |

**重置规则**：每波开始时，卡包等级重置为普通

### 5.2 卡包价格机制（自走棋式）

| 属性 | 数值 |
|------|------|
| 初始价格 | 5金币 |
| 价格递增 | 每次购买+2金币 |
| 重置时机 | 每波开始时重置为5金币 |
| 卡包内容 | 3张随机卡牌 |

### 5.3 卡牌类型

#### A. 永久Buff类（持续整场）

| 卡牌 | 图标 | Lv1效果 | Lv2效果 | Lv3效果 |
|------|------|---------|---------|---------|
| **火力强化** | ⚔️ | 伤害+20% | 伤害+50% | 伤害+120% |
| **急速射击** | ⚡ | 攻速+15% | 攻速+37% | 攻速+90% |
| **射程扩展** | 🔭 | 射程+25% | 射程+62% | 射程+150% |
| **赏金猎人** | 🪙 | 金币+30% | 金币+75% | 金币+180% |

#### B. 即时效果类（一次性）

| 卡牌 | 图标 | Lv1效果 | Lv2效果 | Lv3效果 |
|------|------|---------|---------|---------|
| **全屏轰炸** | 💥 | 100伤害 | 250伤害 | 600伤害 |
| **时间冻结** | ❄️ | 冻结3秒 | 冻结7秒 | 冻结15秒 |
| **紧急维修** | 💖 | 恢复5生命 | 恢复12生命 | 恢复30生命 |
| **群体减速** | 🐌 | 减速30%持续5秒 | 减速50%持续8秒 | 减速70%持续15秒 |

### 5.4 合成系统

```
合成规则：2张相同等级相同类型的卡 → 1张高一级卡

例：
⚔️Lv1 + ⚔️Lv1 = ⚔️Lv2
⚔️Lv2 + ⚔️Lv2 = ⚔️Lv3（最高等级）

合成收益：
Lv1→Lv2：效果提升2.5倍（而非2倍）
Lv2→Lv3：效果提升2.4倍（鼓励合成到满级）
```

### 5.5 卡牌使用

| 模式 | 说明 |
|------|------|
| **手动模式**（默认） | 点击卡牌使用，拖拽合成 |
| **自动模式**（可选） | 获得卡牌立即自动使用 |

**携带上限**：6张卡牌，满后需使用或合成才能继续购买

## 六、横向支援角色系统

### 6.1 位置与机制

- **槽位数量**：4个（左侧2个，右侧2个）
- **攻击方向**：左侧向右，右侧向左

## 七、赌博系统 ⭐新增

### 7.1 解锁条件

```
当前金币 ≥ 卡包价格 × 2

例：卡包价格5g时，金币≥10g解锁赌博
     卡包价格7g时，金币≥14g解锁赌博
```

### 7.2 赌博玩法：幸运转盘

**界面**：3个选项的轮盘赌

| 选项 | 赔率 | 概率 | 说明 |
|------|------|------|------|
| **保守** | 1.5倍 | 70% | 小额稳赚 |
| **平衡** | 2倍 | 50% | 风险收益平衡 |
| **激进** | 3倍 | 30% | 高风险高回报 |

### 7.3 赌博流程

1. 满足解锁条件后，赌博按钮亮起
2. 选择投注金额（最小=卡包价格，最大=当前金币50%）
3. 选择风险等级（保守/平衡/激进）
4. 点击赌博，播放动画
5. 结果：赢取金币 或 损失投注

### 7.4 策略价值

- **滚雪球**：连胜可快速积累金币购买高级卡包
- **翻盘机会**：劣势时搏一把可能逆转
- **风险管控**：可选择保守选项稳赚小额

## 十一、设计分析与优化

### 11.1 已识别的不合理之处

| 问题 | 影响 | 优化方案 |
|------|------|---------|
| **卡包等级只升不降** | 后期过于简单 | 每波重置，保持每波决策重要性 |
| **赌博无冷却** | 可能无限刷金币 | 每波限赌3次，或增加失败惩罚 |
| **自动模式过强** | 降低策略深度 | 自动模式有延迟，或无法合成 |
| **武器更换无成本** | 降低策略选择 | 更换消耗少量金币或有冷却 |
| **卡牌上限固定** | 后期卡手 | 可通过Buff增加卡槽上限 |
| **支援角色性价比** | 法师40g仅3发可能不值 | 增加伤害或降低价格 |

### 11.2 优化后的关键数值

```
卡包等级重置：每波开始时
赌博限制：每波最多3次
自动模式：使用延迟0.5秒，无法自动合成
武器更换：免费但有3秒冷却
卡槽上限：基础6个，可通过特殊Buff+2
法师调整：伤害80→100，或价格40→35
```

### 11.3 平衡性预期

- **前期（0-1分钟）**：专注抽卡积累，建立基础防线
- **中期（1-2分钟）**：卡包等级提升，开始合成高级卡
- **后期（2-3分钟）**：高级Buff叠加，应对高强度波次
- **赌博定位**：锦上添花而非雪中送炭，高风险玩法
